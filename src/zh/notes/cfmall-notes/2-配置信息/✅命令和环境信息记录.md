## 命令

**在Windows环境下的命令窗口操作以下命令**

| **描述** | **命令** |
| --- | --- |
| Windows环境开启虚拟机 | vagrant up |
| 与虚拟机建立连接 | vagrant ssh |
| 查看docker下的服务 | sudo docker ps |
| mysql开机启动 | sudo docker update mysql --restart=always |
|  |  |
